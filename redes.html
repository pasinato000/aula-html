REDE DE COMPUTADORES

Uma rede de computadores é um conjunto de equipamentos interligados de maneira a trocarem informações e compartilharem recursos, como arquivos de dados gravados, impressoras, modems, software e outros equipamentos.

IMPORTÂNCIA DAS REDES DE COMPUTADORES

Proporcionam economia, compartilhamento de recursos, como de impressoras e projetores, bem como de serviços, que é o caso da internet. Permite também, por meio de compartilhamento de arquivos, o trabalho em equipe dos colaboradores.

LAN WAN

LAN (local area network) é uma rede de computadores que cobre uma área limitada, geralmente um edifício ou um campus. É usada para cnectar dispositivos em um espaço fisíco limtado, como uma casa, escritório ou sala de aula. Já WAN (wide area network) é uma rede de computadores que abrange uma área geográfica maior, geralmente uma cidade, um estado ou até mesmo um país,. Ela é usada para conectar dispositivos que estão fisicamente distantes entre si.
